# Coaching System

## Context

Coaching is how Easier turns data into action without shame.

## Problem

Too much feedback becomes noise. Too little feedback leaves the user alone with numbers.

## Solution

Deliver one strong daily suggestion and one weekly summary built from deterministic signals, with AI used to adapt tone and phrasing.

## Output

### V1 coaching objects

- `DailyInsight`: one highest-leverage action
- `WeeklySummary`: recap of wins, friction, hunger patterns, and next focus
- `CoachTone`: supportive, balanced, direct

### Acceptance criteria

- Daily coaching reflects logged behavior and satiety patterns.
- Weekly summaries identify where calories can be saved and hunger can be reduced.
- The same insight payload can be rendered in multiple tones.
