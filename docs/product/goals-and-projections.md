# Goals and Projections

## Context

Goals and projections answer whether the user’s current behavior is actually moving them toward the outcome they want.

## Problem

Users do not just need current numbers. They need a believable timeline and a clear sense of what small changes would do.

## Solution

Use deterministic projections from observed trend data and adherence inputs, then let AI explain the scenario in plain language.

## Output

### User value

- know where they are headed
- see whether the current pace is sustainable
- compare current path versus small adjustments

### Scope

- weight target and optional body-fat target
- smoothed trend calculations
- baseline projection
- one or more adjustment scenarios
- explanation layer for plain-language interpretation

### Acceptance criteria

- Users can set a target and see a baseline projected timeline.
- The system can show at least one alternative scenario with a deterministic adjustment.
- AI does not generate the projection values.
